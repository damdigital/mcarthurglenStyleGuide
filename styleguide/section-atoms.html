1.15.2 - Border utilities

Source: atoms/modifiers/_border.scss, line 87

Open in new window

These classes allow to set a border

The classes are named using the format has-b{sides} for xs and has-b{sides}-{breakpoint}-{none} for the other breakpoints.

Parameter:

  • sides - Optional value. If no set then the 4 sides will be targeted, x target left and right, y top and bottom, r right, l left, t top, b bottom.
  • none - Optional value. Remove border from a set breakpoint

Usage:

  • has-bt-3 => has border top;
  • has-b-sm => has border on all sides from sm breakpoint
  • has-by-lg => has border top and bottom from lg breakpoint
  • has-by-lg-none => remove border top and bottom from lg breakpoint

Example:

Age, inquies, ista parva sunt. An vero, inquit, quisquam potest probare, quod perceptfum, quod. Non potes, nisi retexueris illa. De hominibus dici non necesse est. Nam quid possumus facere melius? Erit enim mecum, si tecum erit.

  • size - Size from the map $spacers from 0 to 6
  • sides - Optional value. If no set then the 4 sides will be targeted, x target left and right, y top and bottom, r right, l left, t top, b bottom.
