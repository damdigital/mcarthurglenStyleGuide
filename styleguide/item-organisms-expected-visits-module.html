3.4 - Expected Visits

Source: organisms/_expected-visits-module.scss, line 39

Open in new window

This module use chart.js and axios to request and display data.

This module do not work in this documentation as we can not make requests to server for security reasons. The graph is replace by an image for illustration purpose.

Popular Times
